◆ write_vec()

int write_vec ( int fd,
struct iovec * iov,
ssize_t towrite,
int vecs )
static

Write "towrite" bytes using writev on non-blocking fd

Every short write is considered an error, transmit can handle that.

Parameters
fd(r) socket fd which must be non-blocking
iov(r) iovec for writev
towrite(r) number of bytes in all iovec elements
vecs(r) number of iovecs in array
Returns
"towrite" bytes written or -1 on error
